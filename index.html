def is_palindrome(s):
    # 移除所有非字母数字字符并转换为小写
    s = ''.join(c.lower() for c in s if c.isalnum())
    
    # 比较字符串与反转后的字符串
    return s == s[::-1]

# 示例用法
print(is_palindrome("A man, a plan, a canal: Panama"))  # 输出: True
print(is_palindrome("race a car"))  # 输出: False

# CodeCraft 贡献指南

感谢你对 CodeCraft 项目的兴趣和贡献！我们欢迎所有开发者的参与，无论经验水平如何。

## 贡献类型

我们接受各种类型的贡献：

- 修复 Bug
- 实现新功能
- 改进文档
- 优化性能
- 提交问题报告
- 提供建议和反馈

## 开发环境设置

1. Fork 仓库到你的 GitHub 账户
2. 克隆你的 fork 到本地：
   ```bash
   git clone https://github.com/your-username/codecraft.git
   cd codecraft
   ```
3. 创建虚拟环境并安装依赖：
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
4. 运行测试确保一切正常：
   ```bash
   pytest
   ```

## 贡献流程

1. 在 GitHub 上查看开放的 Issues，或创建新的 Issue
2. 创建新分支：
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. 进行修改并添加测试
4. 提交修改：
   ```bash
   git add .
   git commit -m "Add your commit message here"
   ```
5. 推送分支到你的 GitHub：
   ```bash
   git push origin feature/your-feature-name
   ```
6. 在 GitHub 上创建 Pull Request

## 代码规范

- 使用 Black 格式化代码
- 使用 Flake8 检查代码风格
- 添加适当的类型提示
- 编写清晰的文档字符串
- 添加单元测试覆盖新功能或修复

## 行为准则

请遵循我们的行为准则：
- 尊重他人的意见和贡献
- 保持礼貌和专业
- 不进行歧视或骚扰行为

## 获得帮助

如果你有任何问题或需要帮助，请在 GitHub Issues 中提问或加入我们的 Discord 社区。
